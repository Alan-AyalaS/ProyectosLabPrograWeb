Etiqueta <header>

Se hizo uso de la etiqueta <header> para contener el navegador de la página. De hecho se puede apreciar que es casi idéntica a la de la página actual, ya que fue reciclada.

Etiqueta <nav>

En esta misma sección de header usamos la etiqueta nav, para dejar en claro que se el código dentro de estas etiquetas es para el desarrollo del navegador ddel sitio web.

Etiqueta <img>

Lo primero que podemos ver en el navegador de izquierda a derecha es el logo de la UEFA Champions League, que incrustamos dentro de las etiqueta <nav> con la etiqueta <img>

Etiqueta <p>

Dentro de las etiqueta <a> usamos la etiqueta <p> para escribir el texto "Lab. Programación Web" y "Práctica 5" que se ve en el extremo izquierdo del navegador.

Etiqueta <a>

Al igual que la etiqueta <p>, dentro de la etiqueta <nav> usamos la etiqueta <a> para color el enlace a esta página en la que se encuentra ahora mimos y la página de inicio donde se encuentran las demás prácticas del laboratorio.

Etiqueta <h1>

Inmediatamente después del navegador pusimos un h1 para presentar el tema de la página. La UEFA Champions League

Etiqueta <audio>

Para poder insertar un audio en el sitio web dimos uso a la etiqueta de <audio>

Etiqueta <section>

Para dividir las secciones dentro de la página usamos la etiqueta section para contener cada una de las secciones. En la imagen se puede ver una de ellas, también se encuentra una sección para una lista de campeones, y además una tabla de las finales.



Etiqueta <article>

Para dvidir distintos elementos dentro de las secciones de la página usamos la etqiueta <article> al dividir los elementos con esta etiqueta podemos manipular más facilmente cada elemento sin afectar al otro.

Etiqueta <table>

Para hacer la tabla de los partidos finales de los últimos 13 años de la copa, hicimos uso de la etiqueta <table> y de sus etiquetas <thead>, <tr>. <th>, <tbody>
